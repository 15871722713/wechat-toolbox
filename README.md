# wechat-toolbox
WeChat toolbox.

# Features

## wxContact.py
* 导出微信通讯录好友（昵称、微信号、备注名、性别、省份、城市、签名）
* 隐私设置缘故，部分微信号无法获取

## wxSendMsg.py
* 向好友或微信群发送文本、图片消息
* 只能读取已保存至通讯录的群
* 连续发送约 100 次后，会受到限制，约 1 小时后才能复活正常
